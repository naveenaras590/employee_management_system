# employee_management_system

Employee Management System

📌 Description

A console-based Employee Management System built using Java and Hibernate ORM to manage employee records with full CRUD operations.

⚙️ Technologies Used

- Java
- Hibernate ORM
- JPA (Annotations)
- PostgreSQL

🚀 Features

- Add, update, delete, and view employees
- Database integration using PostgreSQL
- Layered architecture (Entity, DAO, Service)

🧠 Concepts Used

- Object Relational Mapping (ORM)
- JPA annotations (@Entity, @Id, @Table)
- Hibernate methods: persist(), find(), merge(), remove()

📊 Future Improvements

- Add GUI (Spring Boot / Web)
- Add authentication system
